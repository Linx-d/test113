## git中的忽略文件
- .gitignore,在这个文件中可以设置要忽略的文件或者目录。
- 被忽略的文件不会被提交到仓储中去。
- 在.gitignore中可以书写要被忽略的文件的路径，以/开头，一行写一个路径，这些路径所对应的文件都会被忽略，不会被提交到仓储中
    + 写法 
        * `/.idea` 会忽略.idea 文件
        * `/js` 会忽略js目录中的所有文件
        * `/js/*.js` 会忽略js目录下所有js文件
